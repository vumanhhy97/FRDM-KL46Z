The platinum project supports the following boards and platforms:


Tower cards:
- TWR-KL46Z48M

In order to switch to a different board:

1. Select the project for the overall platform  tower
2. J10[2-3] and J11[2-3] 
3. Connect to a terminal 115200,8,n,1
4. Attach TWRPI-sLCD to J2
5. Download and run the code

Code description.

    A message with a counter will be displayed on TWRPI-sLCD and count increases every time a character is received from terminal.
